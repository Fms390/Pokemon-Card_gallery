# Pokemon Cards Gallery

A simple HTML and CSS project displaying Pokemon character cards in a responsive grid layout.

## Description

This project showcases four Pokemon cards with images arranged in a clean grid design. Built using pure HTML and CSS to practice layout techniques and responsive design.

## Features

- Clean card-based layout
- Responsive image display
- Simple and elegant design
- Pure HTML/CSS (no frameworks)

## Preview

The project displays 4 Pokemon-themed cards with images from various Pokemon characters and scenes.

## Technologies Used

- HTML5
- CSS3

## How to View

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it!
